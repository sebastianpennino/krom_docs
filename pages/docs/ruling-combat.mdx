# Tratado de combate

Mientras la aventura se desarrolla se dice que los jugadores se encuentran en un
entorno narrativo, en este caso las acciones se resuelven como mejor prefiera el
director de juego (usando las tablas pertinentes, improvisando o modificando reglas
sobre la marcha). Cuando los personajes están en una situación de combate se dice
que el juego se resuelve en un entorno táctico. En el entorno táctico las reglas
son mucho más específicas y precisas, todo el «tratado de combate» describe con
el mayor detalle posible lo que sucede en el entorno táctico.

 
```json filename="1.json"
{
  "01": "1- Secuencia de combate",
  "02": "2- Modificadores comunes",
  "03": "3- Acciones comunes",
  "04": "4- Ataques",
  "05": "5- Maniobras",
  "06": "6- Criticos",
  "07": "7- Pifias"
}
```
