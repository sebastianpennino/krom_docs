# Individuales
Las habilidades "sueltas" deben ser escogidas con cuidado ya que avanzan de forma muy lenta (requieren cuatro skillchecks para avanzar envés de tres que es lo normal en habilidades de skillpacks) Cada habilidad pertenece a una categoría o grupo de habilidades, esta categoría por lo general determina como se calcula el valor inicial de dicha habilidad

<Callout type="info">
  Las habilidades individuales requieren 4 skillchecks para avanzar
</Callout>

<Callout type="error">
  Toda esta sección - especialmente las fórmulas - son de una version anterior
  de Kromsys, seguramente cambien en breve. 
</Callout>
