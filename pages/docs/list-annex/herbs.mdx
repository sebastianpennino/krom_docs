# Hierbas (plantas medicinales)

Este anexo funciona como un generador de hierbas, util tanto para el director como para personajes.
En caso de querer usarlo como personaje, preguntar al director de antemano.

Para usarlo:

1. Primero determina **Aplicacion / modo de administración** (a elección o tirando 2d10 y sumando resultado)
2. Luego determina **que forma tiene o que parte de una planta es** (a elección o tirando 3d10 y sumando el resultado)
3. Finalmente determinamos nombre, como buscar la hierba y que hace usando **dado porcentual** repetidas veces.

<Callout type="info">
  El dado porcentual es tirar d10 como decena y otro d10 como unidad
</Callout>

Tirando el dado porcentual:

> - Lanza d10 avisando que será la decena (multiplica x10 el resultado, trata los ceros como ceros)
> - Lanza otro d10 de unidad (trata los ceros como "diez")
>
> Así por ejemplo: obtenitendo 7 en decena y 0 en unidad -> 70 + 10 -> 100

Usa este procedimiento para primero determinar el nombre usando una tirada para Prefijo y otra para Sufijo.
Luego otra tirada más para determinar "Cuando intentes encontrar, busca..."
Y finalmente elige Efectos A o B antes de tirar, y vuelve a tirar un porcentual para saber los efectos de la hierba.
