# Equipo inicial

En base al skillpack elegido obtienes un set de items iniciales

## Aristocrata / Academico / Comerciante / Cortesano

- Vestimenta: ropas distinguidas, guantes, capa, botas de cuero, sombrero
- Espejo de mano (5x5 cm)
- Tintero, tinta y pluma para escribir
- Cantimplora (cuerno de marfil)
- Linterna de aceite

## Herborista / Arqueologo / Campesino

- Vestimenta: ropas de trabajo, botas de cuero endurecidas, guantes gastados, sombrero
- Cantimplora de cuero
- Pala con punta metálica
- Tijeras de podar si es herborista o campesino, pico de escalar si es arqueólogo
- Soga común (10m)
- Riñonera

## Artesano

- Vestimenta: ropas de trabajo, botas de cuero endurecidas, guantes de cuero curtido
- Cincel
- Martillo
- Balde
- Serrucho

## Ladronzuelo / Infiltrador

- Vestimenta: ropas urbanas, botas de cuero endurecidas, guantes, capucha, capa
- Pico para escalar
- Soga común (10m)
- Linterna de aceite
- Cama enrollable

## Ritualista

- Vestimenta: túnica con capucha, botas de cuero, guantes negros
- Tintero, tinta y pluma para escribir
- Velas
- Tiza
- Pedernal y eslabón (para prender fuego)

## Soldado / Marinero / Luchador

- Vestimenta: ropas urbanas, botas con espuelas, cinturón, guantes
- Cama enrollable
- Cantimplora de cuero
- Pala con punta metálica si es soldado / Balde si es marinero
- Soga común (10m)

## Entretenedor

- Vestimenta: ropas distinguidas, capa, botas de cuero, sombrero
- Instrumento a elección
- Espejo de mano (5x5 cm)
- Kit de maquillaje
- Riñonera
