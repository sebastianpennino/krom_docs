# Comprando Historiales

**El historial** puede ser visto en la sección especifica de la **subclase ya elegida**, 
por ejemplo este es el [Historial del Cazamagos](/docs/list-classes/w-mage/mage-hunter#historial).

El historial de cada subclase es radicalmente diferente y cada director puede
imponer las modificaciones que crea necesarias (reducciones o aumentos de
costes, agregados), mientras les avise a los jugadores con anticipación
(al menos antes de elegir prioridades).

| Prioridad |            Puntos y dinero            |
| :-------: | :-----------------------------------: |
|     A     | 14pt de historial y 100mc para equipo |
|     B     | 9pt de historial y 100mc para equipo  |
|     C     |  7pt de historial y 50mc para equipo  |
|     D     |  5pt de historial y 50mc para equipo  |
|     E     |  2pt de historial y 25mc para equipo  |

<Callout type="info" emoji="✏️">
  {" "}
  Básandote en la prioridad, elige tu historial tratando de que no queden puntos
  sin utilizar, ya que se pierden.
</Callout>
