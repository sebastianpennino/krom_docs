# Protecciones

## Descripción

Como [Curaciones](/docs/list-hexagons/e18) pero en vez de curar puede usarse para colocar un “escudo magico de protección” que recibe daño antes que la vida del objetivo

## Específicos

- Cuando se usa como curar, sigue las reglas ya establecidas.
- Cuando se usa como protección usa esta fórmula: `Mística + 3 puntos por éxito`
- El resultado es una “protección mágica” que se coloca sobre el objetivo (un objetivo no puede tener más de una protección mágica activa a la vez) que debe destruirse primero antes de poder dañar al objetivo.

## Resistencias

No hay tiradas de resistencia

## Utiliza

Mismo calculo que un ataque mágico pero usando: `Mística + 3 puntos por éxito`

## Guia de dificultad

No aplica ya que se calcula como ataque
