# Sonido (sonomancia limitado)

## Descripción

Como Sonomancia pero limitado (ver limitaciones)

## Limitaciones

- Duración máxima: fugaz (30s)
- Alcance máximo: voz (10m)
- No puede usar objetivos

## Ejemplos de uso

Ver los ejemplos de Sonomancia pero usar las limitaciones impuestas

## Resistencias, Utiliza, Guía de dificultad

Seguir las recomendaciones de sonomancia
