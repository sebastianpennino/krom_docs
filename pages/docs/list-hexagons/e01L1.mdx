# Llamas (fuego limitado)

## Descripción

Como fuego (magias elementales) pero muy limitado

## Limitaciones

- Solo se usa en modo narrativo (no puede usarse como ataque mágico)
- Duración máxima: fugaz (30s)
- Alcance máximo: voz (10m)
- Volumen máximo: habitacion (5m)

## Ejemplos de uso

- Prender un pequeño fuego entre las manos para ver de noche
- Encender una fogata
- Aumentar el calor del fuego del hogar de leña
- Generar un pequeño muro de fuego
