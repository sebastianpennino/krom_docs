# Curaciones

## Descripción

Es la capacidad mágica de curar heridas y restaurar puntos de vida.

## Específicos

La curación de vida se toma como un hechizo ofensivo pero que en vez de dañar, repara (calcular el hechizo como si fuese un ataque mágico, pero utiliza el valor obtenido para curar).
Se pueden sacrificar dados con ceros (no se cuentan para el valor a curar) para subsanar efectos. Por ejemplo, un cero remueve los efectos de un crítico típico (por ejemplo, quitan conmoción o sangrado) dos ceros remueven un crítico atípico y así sucesivamente.

## Resistencias

Si se está curando a un objetivo que intenta resistirse (sería un caso rarisimo, ver con director) se debe superar la resistencia mágica del objetivo

## Utiliza

Mismo calculo que un ataque mágico

## Guía de dificultad

No aplica ya que se calcula como ataque
