# Curar
## Descripción
Es la capacidad mágica de curar heridas y restaurar puntos de vida.
## Específicos
La curación de vida se toma como un hechizo ofensivo pero que en vez de dañar, repara (calcular el hechizo como si fuese un ataque mágico pero utiliza el daño obtenido para curar). 

Corregir: solo cancela criticos con ceros.

Se pueden sacrificar dados (no se cuentan para el valor del ataque mágico) para curar efectos, por ejemplo un cero o dos éxitos curan un crítico típico (por ejemplo, quitan conmoción o sangrado) dos ceros o cuatro éxitos remueven un crítico atípico.
## Resistencias
Si se está curando a un objetivo que intenta resistirse (sería un caso rarisimo, ver con director) se debe superar la resistencia mágica del objetivo
## Utiliza
Mismo calculo que un ataque mágico
## Guia de dificultad
No aplica ya que se calcula como ataque
